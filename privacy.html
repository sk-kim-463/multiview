개인정보처리방침 (Privacy Policy)

Effective Date / 시행일자: June 10, 2026

1. Overview / 개요

EN: This Privacy Policy describes how this service (the "Service") handles data when utilizing the YouTube Data API v3. We deeply value user privacy and strictly comply with all relevant global and domestic data protection regulations.
KR: 본 개인정보처리방침은 유튜브 API(YouTube Data API v3)를 이용하는 과정에서 어떠한 데이터가 어떻게 처리되는지 설명합니다. 본 서비스(이하 "서비스")는 사용자의 개인정보를 중요시하며, 관련 법령을 준수합니다.

2. Collection of Personal Information / 수집하는 개인정보 항목 및 방법

EN:
  • No PII Collection: The Service does not request, collect, or record any Personally Identifiable Information (PII) such as names, email addresses, phone numbers, or passwords.
  • API Data Fetching: The Service only retrieves publicly available real-time streaming metadata (such as concurrent viewer counts) directly from the YouTube Data API via the user's web browser.
KR:
  • 개인식별정보 수집 불가: 본 서비스는 사용자의 이름, 이메일, 비밀번호 등 어떠한 개인 식별 정보도 직접 수집하거나 요구하지 않습니다.
  • API 데이터 조회: 본 서비스는 유튜브 API를 통해 공공에 공개된 실시간 스트리밍 정보(시청자 수 등)만을 브라우저 상에서 일시적으로 조회합니다.

3. Purpose of Data Usage / 데이터 이용 목적

EN: Any metadata retrieved via the YouTube Data API is utilized exclusively to display the 'Real-time YouTube Concurrent Viewers Dashboard' functionality on the screen dynamically. No data is used for commercial profiling or advertising tracking.
KR: API를 통해 조회한 데이터는 오직 사용자가 웹 브라우저에서 '실시간 유튜브 시청자 수 대시보드' 기능을 이용할 수 있도록 화면에 표시하는 목적으로만 사용되며, 마케팅이나 추적 광고 등 다른 목적으로는 절대 활용되지 않습니다.

4. Data Retention and Destruction / 데이터의 보유 및 파기

EN:
  • Zero Serverless Footprint: The Service is engineered as a serverless static application. We do not operate or maintain any backend servers, log collectors, or external database systems.
  • Instant Destruction: All retrieved API payloads reside temporary within the volatile browser memory (RAM) and are instantly, completely, and permanently destroyed as soon as the user closes the browser window or tab.
KR:
  • 백엔드 서버 미운영: 본 서비스는 자체 백엔드 서버나 DB 로그 시스템을 전혀 운영하지 않는 정적 서버리스 구조입니다.
  • 즉시 파기: API로 가져온 모든 데이터는 사용자의 웹 브라우저 휘발성 메모리(RAM) 내에서만 처리되며, 브라우저 창이나 탭을 닫는 즉시 완전히 자동으로 파기됩니다.

5. Third-Party Sharing / 제3자 제공 및 위탁

EN: Since the Service does not collect, capture, or store any personal or technical metadata, there is no technical possibility of disclosing, sharing, or selling user data to third-party entities.
KR: 본 서비스는 수집 및 저장하는 데이터 자체가 없으므로, 이용자의 개인정보를 외부에 제공, 공유 또는 위탁할 수 없으며 실제로 수행하지도 않습니다.

6. Compliance with Google API Policies / 구글 사용자 데이터 정책 준수

EN: The Service strictly adheres to the Google API Services User Data Policy, including the Limited Use requirements. We maintain complete transparency and never breach any guidelines enforced by Google Cloud and YouTube developer terms.
KR: 본 서비스는 제한적 사용 요구사항을 포함한 Google API 서비스 사용자 데이터 정책을 엄격히 준수합니다. 구글 클라우드 및 유튜브 개발자 약관이 요구하는 보안 및 투명성 기준을 무조건 이행합니다.
